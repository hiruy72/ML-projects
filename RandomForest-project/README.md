# Simple Classification Project

Build a classification model using RandomForestClassifier with the standard ML workflow.

## Quick Start
```bash
pip install -r requirements.txt
python main.py
```

## The 6-Step Workflow
1. **Load Data** - Iris dataset (150 flowers, 4 features)
2. **Preprocess** - Clean data (Iris is already clean!)
3. **Split Data** - 80% training, 20% testing
4. **Train Model** - RandomForestClassifier
5. **Make Predictions** - Predict flower species
6. **Evaluate** - Check accuracy and performance

## What You'll Learn
- Complete ML workflow with scikit-learn
- RandomForestClassifier for classification
- Model evaluation with accuracy and classification report
- Feature importance analysis

## Expected Results
- High accuracy (usually 95%+ on Iris dataset)
- Clear classification report showing precision, recall, F1-score
- Feature importance rankings
